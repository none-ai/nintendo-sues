About This Project

Project Information

This is a Flask web application providing comprehensive coverage of Nintendo's lawsuit against the US government regarding tariff classifications on video game consoles and accessories.

Features

  • Home Page: Overview of the lawsuit and key developments
  • Full Article: In-depth coverage with background and analysis
  • Timeline: Chronological events of the legal battle
  • Discussion: Comment section for reader engagement
  • Contact: Feedback form for user input
  • Dark Mode: Toggle between light and dark themes

Tech Stack

  • Python
  • Flask
  • HTML5 / CSS3
  • JavaScript
